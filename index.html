About Me

Building with purpose.

I'm an Android developer focused on building apps that are genuinely useful — clean architecture, smooth UX, and strong privacy practices. I believe software should solve problems without creating new ones. Every app I build works offline, requires no account, and never sells user data.

🏗️ How I Build

Clean Architecture with strict MVVM separation. Domain, data, and presentation layers stay independent. Code is testable, maintainable, and scalable from day one.

🎯 What I Value

Simplicity over feature bloat. Offline-first design. User privacy by default. Fast, intuitive UX that gets out of the way and lets users focus on what matters.

📱 Platform Focus

Android-native with Kotlin and Jetpack Compose. I build for Android 7+ and follow Material Design principles with custom UI components.

🔧 Tech Stack

Kotlin · Jetpack Compose · Room · Koin · Coroutines · Flow · Firebase · AdMob · WorkManager · DataStore
