About

Building with purpose.

Senior Android developer with 7+ years of experience building high-quality, scalable mobile applications using Java and Kotlin. Proficient in Jetpack Compose for modern UI design, with deep expertise in Coroutines, Flow, and Paging 3 for efficient asynchronous data handling. Strong backend integration skills across RESTful APIs, GraphQL, and Firebase. I believe software should solve real problems without creating new ones — clean architecture, smooth performance, and long-term maintainability are non-negotiable.

🏗️

Architecture First

MVVM and MVI with Clean Architecture. Domain, data, and presentation layers stay strictly separated. Dependency injection via Hilt and Koin. Testable and scalable from day one.

⚡

Performance Focused

Kotlin Coroutines and Flow for async operations. Paging 3 for efficient data loading. WorkManager for background tasks. Optimized builds that are fast, responsive, and battery-friendly.

🔗

Backend Integration

Retrofit for RESTful APIs, GraphQL for flexible queries, Socket.IO for real-time communication. Payment integrations: Stripe, PayPal, Google In-App Billing, and Google Subscriptions.

📱

Modern Android

Jetpack Compose with Material Design 3. Navigation Component, Room, DataStore, Firebase suite. Offline-first architecture with local-first data strategies.
