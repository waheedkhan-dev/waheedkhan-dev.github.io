About

Building with purpose.

I'm an Android developer who builds apps that respect users — no unnecessary accounts, no cloud data collection, no dark patterns. My philosophy is simple: if a feature doesn't make the app meaningfully better for the user, it doesn't belong. I focus on clean architecture, smooth performance, and long-term maintainability.

🏗️

Architecture First

Clean Architecture with strict MVVM separation. Domain, data, and presentation layers stay independent. Testable and scalable from day one.

🔒

Privacy by Default

No unnecessary data collection. Apps work offline, require no account, and store data locally. User trust is never traded for analytics.

📱

Android Native

Kotlin and Jetpack Compose, targeting Android 7+. Material Design principles with custom UI components that feel native and fast.
