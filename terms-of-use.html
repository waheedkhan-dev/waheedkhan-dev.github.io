2. Description of Service

Routinic is an Android habit tracker that helps you build and maintain positive daily routines. The app provides:

  • Create and manage daily, weekly, monthly, and one-time habits using custom settings or pre-built templates
  • Track habit completion, build streaks, and earn Awesome Days
  • View progress through bar-chart analytics, a monthly calendar heatmap, and per-habit statistics
  • Set up to three reminders per habit with exact-time alarms
  • Run timed habits — meditation, focused work, workouts — using the built-in countdown timer
  • Break bad habits using the dedicated Quit Bad Habits category
  • Customise the app with Light, Dark, and Dark Blue themes
